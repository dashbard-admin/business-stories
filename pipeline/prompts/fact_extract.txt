You are extracting verifiable facts about {incident_name} from a single
source. Output ONLY facts that the source explicitly states. Do NOT
infer, embellish, or summarize beyond what is written.

For each fact, output an object with these fields:

- fact_text: a single declarative sentence
- fact_type: one of [
    founding_date,
    location_founded,
    founder,
    early_employee,
    product_launch,
    business_model,
    market_context,
    competitor,
    crisis_trigger,
    pivotal_decision,
    financial_metric,
    funding_round,
    acquisition,
    ipo_event,
    regulatory_event,
    lawsuit_or_settlement,
    leadership_change,
    quote,
    documentary_evidence
  ]

  fact_text should name the document AND state what it shows. These
  facts feed S06's Act 3.5 "Investigation" slot — that act is meant
  to walk the viewer through how-we-know-what-happened, and without
  documentary_evidence facts in the ledger the investigation slot

  otherwise.

  second paragraph", "section titled 'Founding'", "paragraph
  beginning 'In 2003...'")
- confidence: "high" if directly stated, "medium" if implied from
  surrounding context, "low" if requires interpretation
- hq_location (OPTIONAL — only include if the source EXPLICITLY

  "headquartered in Palo Alto, California" or "based in London").
  Format as an object: {{"city": "...", "state_or_region": "...",

  surrounding context — leave that to downstream consolidation.

SOURCE METADATA:
